# 🌐 Desarrollo de Aplicaciones Web
## UEA - L-UFPTI-004

Bienvenido al repositorio oficial de la asignatura **Desarrollo de Aplicaciones Web** de la Universidad Estatal Amazónica (UEA).

## 📖 Descripción

Este repositorio contiene materiales académicos, recursos de aprendizaje, actividades, prácticas, proyectos y documentación relacionados con la asignatura de Desarrollo de Aplicaciones Web.

Su objetivo es servir como un espacio centralizado para el acceso a contenidos, ejemplos y recursos que apoyen el proceso de enseñanza y aprendizaje de tecnologías y metodologías utilizadas en el desarrollo de aplicaciones web modernas.

## 🎯 Objetivo

Fortalecer las competencias de los estudiantes en el análisis, diseño, desarrollo e implementación de aplicaciones web mediante el uso de herramientas, tecnologías y buenas prácticas de la industria del software.

## 📂 Contenido del Repositorio

- Material de apoyo.
- Recursos académicos.
- Ejemplos prácticos.
- Actividades y talleres.
- Proyectos de desarrollo.
- Documentación complementaria.

## 🛠 Tecnologías

A lo largo de la asignatura se utilizarán diferentes herramientas y tecnologías relacionadas con el desarrollo web, programación, bases de datos, control de versiones y despliegue de aplicaciones.

## 📚 Uso Académico

Este repositorio tiene fines exclusivamente académicos y está orientado al apoyo de las actividades desarrolladas dentro de la asignatura.
